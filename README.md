# PokeAppi

Proyecto inicial con React, Tailwind CSS y Vite usando `pnpm`.

## Estructura común

- `index.html`
- `src/main.jsx`
- `src/App.jsx`
- `src/index.css`
- `src/components/`
- `public/`

## Comandos

```bash
pnpm install
pnpm dev
pnpm build
pnpm preview
```

## Descripción

Este proyecto usa:

- React 19
- Tailwind CSS 4
- Vite
- `pnpm`
